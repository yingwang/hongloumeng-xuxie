# 红楼梦续写

《红楼梦》后四十回续写。

公开页面：https://yingwang.github.io/hongloumeng-xuxie/

本项目以《红楼梦》前八十回之后为岔口，续写第八十一回至第一百二十回。不是考据校注，也不替代通行程高本；写作目标是承接前八十回的人物性情、诗梦伏线、家事冷暖与盛衰脉络。

## 目录

- `chapters/`：正文，按回存放。
- `notes/outline.md`：总纲和回目规划。
- `notes/characters.md`：人物线索和写法约束。
- `notes/style-guide.md`：章回体、叙述口吻和修订原则。

## 写作原则

- 先接住前八十回的伏线，再推进新情节。
- 不急于兑现大悲局，先写人心各自转向。
- 每回至少有一件府中实事、一处诗梦谶语、一段人物关系变化。
- 文风取章回小说的骨架，不硬摹古腔；叙述要清楚，人物要能行动。
