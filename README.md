# hongloumeng-xuxie

《红楼梦》续写项目。

公开页面：https://yingwang.github.io/hongloumeng-xuxie/

本 repo 以《红楼梦》前八十回之后为岔口，另写一条续书线；不是考据校注，也不替代通行程高本。目标是保留原书的梦、谶、诗、家事、人情四层推进，同时把每回写成可独立修订的文本。

## 目录

- `chapters/`：正文，按回存放。
- `notes/outline.md`：总纲和回目规划。
- `notes/characters.md`：人物线索和写法约束。
- `notes/style-guide.md`：章回体、叙述口吻和修订原则。

## 当前进度

- 旧版第八十一至八十五回已废弃，不再作为当前续写的一部分。
- 重起稿已起草：第八十一回《晒残书潇湘添冷句 议减例怡红识秋声》

## 写作原则

- 先接住前八十回的伏线，再推进新情节。
- 不沿用旧稿的场景、句段和情节机关。
- 不急于兑现大悲局，先写人心各自转向。
- 每回至少有一件府中实事、一处诗梦谶语、一段人物关系变化。
- 文风取章回小说的骨架，不硬摹古腔；叙述要清楚，人物要能行动。
